# Dashboard Icons (MAP - ICONS)

Home screen action-tile icons are the provided SVGs from **D:\MAP - ICONS**, inlined in these files:

- **rector-icons.ts** — Out pass, Leave, Profile, Notice Board (from MAP - rector)
- **warden-icons.ts** — Profile, Attendance, Checklist, Requests, Students, Notice Board (from MAP -warden)
- **guard-icons.ts** — Profile, QR Scan, Checklist, Notice Board, Guest Entry, Leave, Out pass (from MAP - security guard)
- **campus-manager-icons.ts** — Profile, Checklist, Requests, Notice Board, Emergency, My Staff (from MAP - Campus manager)
- **laundry-manager-icons.ts** — Profile, Raise Request, Active Request, Notice Board (from MAP - Laundry)
- **sports-manager-icons.ts** — Profile, Active Request, Notice Board, Courts List (from MAP - Sports manager)

The `source/` folder holds a copy of the provided SVGs for reference. To change an icon, replace the corresponding `String.raw\`...\`` value in the right `*-icons.ts` file with the new SVG markup.
