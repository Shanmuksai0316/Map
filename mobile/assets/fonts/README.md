# Custom fonts (staff header)

**Ethnocentric Rg** is used for the staff app header "KARTA" text.

- `EthnocentricRg.otf` – linked via `react-native.config.js` assets.
- In code use: `fontFamily: 'EthnocentricRg'`.
- Rebuild the app after adding or changing fonts: `npx react-native run-android` (or run-ios).
